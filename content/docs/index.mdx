有關MIG（多實例GPU）的詳細資料如下：

# 啟用MIG模式

- 使用`nvidia-smi`命令在指定的GPU上啟用MIG模式。例如，以下命令啟用MIG模式：
  ```
  $ nvidia-smi -i [gpu-ids] -mig 1
  ```
- 在啟用後，您可以通過查詢MIG模式來確認它是否已啟用 。

## GPU-ID

以逗號分隔的 GPU 索引、PCI 匯流排 ID 或 UUID 列表，指定要啟用 MIG 模式的 GPU。如果省略gpu-ids，則在系統上的所有 GPU 上啟用 MIG 模式。

- 此範例在 GPU 0 上啟用 MIG 模式。
  ```sh
  $ nvidia-smi -i 0 -mig 1
  ```

2. **在MIG啟用的GPU上創建GPU實例**

   - 一旦啟用了MIG模式，您可以創建GPU實例。例如，以下命令列出可用的GPU實例配置：
     ```
     $ nvidia-smi mig -lgip
     ```
   - 然後使用對應的配置ID來創建GPU實例 。

3. **創建計算實例**

   - 在創建了GPU實例後，您可以選擇在這些實例中創建計算實例。使用以下命令來創建計算實例：
     ```
     $ nvidia-smi mig -cci compute-instance-profile-id -gi gpu-instance-id
     ```
   - 創建完成後，可以通過`nvidia-smi`命令來驗證這些實例 。

4. **禁用MIG模式**
   - 如果您希望在某些GPU上禁用MIG模式，可以使用以下命令：
     ```
     $ nvidia-smi -i [gpu-ids] -mig 0
     ```
   - 確保在禁用前，該GPU未被其他進程使用 。
