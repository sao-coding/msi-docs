## 擴充 xrdp session

### 1. 開啟 xrdp 設定檔

在終端機中輸入下方指令：

```bash
sudo vim /etc/xrdp/sesman.ini
```

![](../img/xrdp/xrdp12.png)

### 2. 修改 MaxSessions 數量

1. 輸入 `i` 進入編輯模式
2. 將 `MaxSessions` 後面的數字（預設為 10）改成所需增加的 session 數量

![](../img/xrdp/xrdp13.png)
![](../img/xrdp/xrdp14.png)

### 3. 存檔並離開

1. 點擊 `esc` 鍵進入指令模式
2. 輸入 `:wq!` 並按 `Enter` 鍵存檔並離開

![](../img/xrdp/xrdp15.png)

### 4. 重新啟動 xrdp

在終端機中輸入下方指令：

```bash
sudo service xrdp restart
```

![](../img/xrdp/xrdp16.png)
